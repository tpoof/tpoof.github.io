These SVG graphics were created by hand using inline SVG markup directly inside an HTML document. Each shape uses native SVG elements: <rect> for the rectangle, <circle> for the circle, and <polygon> for the custom shape. The gradient is defined with a <radialGradient> element inside a <defs> block and applied via a fill="url(#id)" reference so color can scale infinitely without any image files. Styling is handled by the shared styles.css file used across all my assignments.

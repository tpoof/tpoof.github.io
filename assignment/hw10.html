Part 2 — JavaScript Question

Given the following JavaScript array of objects:

var friends = [{
    "name": "John",
    "age": "28",
    "gender": "male",
    "profession": "sales rep"
},
{
    "name": "Maryenne",
    "age": "26",
    "gender": "female",
    "profession": "web developer"
}];

And the following code that constructs an HTML table row:

var tablecell1 = '<tr><td>' + friends[0].name +
'</td><td>' + friends[0].profession +
'</td></tr>';

How the expression evaluates:

  • 1
    friends[0] accesses the first object in the array (index 0 = John's object)
  • 2
    friends[0].name returns "John"
  • 3
    friends[0].profession returns "sales rep"
  • 4
    The + operator concatenates the HTML tags and values into one string

Complete value of tablecell1:

Answer — full HTML output
<tr><td>John</td><td>sales rep</td></tr>
